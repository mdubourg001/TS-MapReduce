**This is a TS implementation of the MapReduce algorithm. (Context: EPSI I2 BIG DATA)**

#### Setup project

```shell
yarn # installing dependencies
```

#### Run

Running does not logs results at the moment, it only measure and logs the execution time.

```shell
yarn start
```
